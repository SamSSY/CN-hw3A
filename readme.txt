Q:How to compile?
A: there is a Makefile, so just type "make" in cli
	-----------------------
	$ make
	-----------------------

Q: How to run?
A: After compiling, there would be a executable file 'router', so type the commands in cli:
	-----------------------
	$ ./router [path/input_file_name] [path/output_file_name]
	-----------------------

	ex:
	-----------------------
	$ ./router ../testfiles/in3.txt ../outputs/out3.txt
	-----------------------
